§ 1.B

Open IA questions for this pass

decide before hi-fi · 4 items

Q1 · placement

Ranges under Review or Reports?

Currently in Reports — but it's used during hand review, not after. Recommend moving up to Review next to Hands.

Q2 · shape

Arena: one drill or a hub?

If we list "drills available" as a hub, the user has to pick. If we auto-load the top leak, friction drops. Recommend auto-load with "switch drill ▾".

Q3 · depth

Villains: dossier worth it?

44 villains, but the long tail has <30 hands of evidence. Maybe show only the top 6 as dossiers, the rest as table-only.

Q4 · scope

Career = Dashboard zoomed out?

Risk of redundancy. The differentiator is the narrative: milestones rail, profile evolution. If we can't commit to that, fold Career into Dashboard's lifetime tab.

DIRECTIONS / DASHBOARD V8 / RE-SKINS

Three ways to dress v8.

Same content, same information density, same hierarchy as Dashboard v8 — three different visual systems. The question to answer here is tone, not features. Pick one, then propagate it to the eight pages sketched above.

A · CALM EDITORIAL

Warmer paper, less chrome

Serif headlines, monospace details, a thin gold accent. Reads like a quarterly letter to yourself. Lots of white space, no cards-in-cards.

B · TRADING TERMINAL

All mono, info-dense, mint accent

Bloomberg / hedge-fund desk. Tight grid lines, no rounded corners on data, ASCII curves. For a power user that lives in this tool daily.

C · FELT CASINO

Green felt, gold accents, chips

Leans into the metaphor. The dashboard is the table. Warm and a little theatrical — the chip stack motif from v8 extended across the page.

Recommendation.
Use C · Felt on Dashboard + Arena (the moments the user wants to feel like they're at the table) and B · Terminal on Hands, Ranges, Sessions (the moments the user wants to work). A · Calm Editorial earns Career and Leaks read-out — places where the tone is reflection, not action.

Said another way: one product, three modes — table / desk / desk-side reading. The shared layer is v8's information architecture and color semantics (pos / neg / warn / accent); everything else is mode.
