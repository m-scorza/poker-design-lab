Direction Lab — same dashboard, four voices

Identical content and data in every file (your real product's Dashboard: lifetime result, preflop profile, leaks, sessions, verdict). Only the design language changes. Open each one full-screen, give it ten seconds, note your gut reaction before your analysis kicks in.

00The Slop Baselinethe refusal

Deliberately generic AI-SaaS: Inter, violet gradients, rounded glowing cards, "AI Insights ✨". Competent and anonymous. This is what no-decisions looks like — included so you can see what we're refusing.
01Pure Instrumentlinear / vercel

Flat, dense, monochrome. 1px borders, tiny uppercase labels, tabular numbers, zero motion, color only where it means something. The 4-hour-session screen. Question: does it feel like craft, or like nothing?
02The Ledgereditorial, static

Same quiet structure, but editorial type does the talking: Bricolage display, mono kickers, one monumental number, hairline rules instead of boxes. No cards, no glow, no motion. The lab's best idea, sobered up.
03The Synthesisledger + motion budget

The Ledger plus exactly five motion moments: load choreography, the monument counts up, the arc draws itself, hover micro-interactions, a button that means it. Reload the page to replay. Question: does the motion add conviction, or do you tire of it by the third reload?

How to judge

1 — Reload each a few times. The third impression matters more than the first; animation charms once but you'll see it daily.

2 — Imagine it after a losing session. Which one are you comfortable getting bad news from?

3 — You're allowed to mix: e.g. "01's density with 02's type" or "03 but half the motion." The pick doesn't need a name, it needs your words.

4 — Whatever you decide gets written into DECISIONS.md as the founding choice the real design system is built on.
